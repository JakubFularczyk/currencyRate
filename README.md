# Currency Rate Service

Simple Spring Boot application for fetching and storing currency exchange rates.

## Tech stack
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL

## Features
- Fetch exchange rates from NBP API
- Store rates in database
- REST API for currency conversion

## Note
For demonstration and local testing purposes, this repository includes a `.env` file.
In real-world and production projects, such files should never be committed to version control.
